# Sistema de Registro de Visitantes - POO

Aplicación CRUD con arquitectura modular por capas usando Tkinter (solo librerías estándar).

## Funcionalidades
- Registrar nuevo visitante (cédula única)
- Visualizar lista en tabla (ttk.Treeview)
- Eliminar visitante seleccionado
- Limpiar campos del formulario automáticamente
